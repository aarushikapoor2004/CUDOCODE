# 🚀 Cudocode Analytics Dashboard

## 📌 Overview
Cudocode is a data analytics project focused on analyzing a coding platform dataset to uncover insights related to **user engagement, problem-solving performance, and chatbot impact**.

The project simulates real-world product analytics use cases and demonstrates how data can drive decision-making.

---

## 📊 Dataset
The dataset (~100 records) includes:
- Users
- Submissions
- Problems
- Chatbot interactions

---

## 🎯 Key Objectives
- Analyze user activity and engagement
- Evaluate success vs failure rates
- Understand problem difficulty distribution
- Measure the impact of chatbot usage
- Generate insights for product improvement

---

## 🔍 Key Insights
- 📈 Overall success rate: ~46%
- 🤖 Chatbot usage improves problem-solving success
- 👤 Identified top active users based on submissions
- ⚖️ Balanced distribution across difficulty levels

---

## ⚙️ Features
- Data cleaning and preprocessing using **Pandas**
- Dataset merging and transformation
- KPI generation and summary metrics
- Visualizations using **Matplotlib**
- Exported datasets for dashboarding
- Simple GenAI-style analytics assistant for querying insights

---

## 📊 Visualizations
The project includes:
- Top Active Users chart
- Success vs Failure distribution
- Problem difficulty distribution
- Chatbot impact analysis
- Daily submission trends

---

## 🛠️ Tech Stack
- Python (Pandas, Matplotlib)
- SQL concepts (joins, aggregations)
- Power BI (for dashboard visualization)

## 🚀 How to Run
1. Clone the repository

git clone https://github.com/your-username/cudocode-analytics.git


2. Navigate to project folder

cd cudocode_python


3. Install dependencies

pip install pandas matplotlib seaborn


4. Run the script

python analysis.py
